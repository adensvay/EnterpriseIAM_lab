# EnterpriseIAM_lab


No longer procrastinating this project. We will have some free time soon to work on projects this winter quarter. Baby steps... the goals is to have a nearly finished IAM AD working lab with windows server installed, GPOs and OUs established, and working on linking Azure sync. 

Tier 1 (Intro/Initialization)
- Downloading/Deploy Windows Server on VirtualBox
- Installing MS AD
- Configuring static networking
- Connecting client to domain

Tier 2 (foundational IAM) 
- creating GPO (Group Policy) 
- creating OU (Organizational Units)
- Lifecycle simulation + visualization tool
- Schema on Figma for Visualization tool

Tier 3 (adding flair and advancing project)
- PowerShell automation creating bulk users, importing csv files
- FAQ page on common trouble shoots and soft/hard questions on AD
- Probe cybersecurity professional on more specific questions to add to the FAQ page

Tier 4 (Fancy Stuff + Dev Skills)
- connect with MS Azure Sync
- Security Dashbord UI/UX linking servers live data
